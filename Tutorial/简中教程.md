## 1️⃣ 漫画库页面介绍

<img src="https://raw.githubusercontent.com/jedzqer/manga-translator/main/Tutorial/mangaku-screenshot.jpg" alt="漫画库页面" width="45%" />

漫画库用于 **管理和储存漫画** 📂

* 点击右下角的 **➕ 加号按钮** 可以创建文件夹
* 进入文件夹后，可以添加漫画图片
* ⚠️ **为了保证漫画可以按正确顺序阅读**，请务必确保：

  * 图片的 **文件名顺序** 与漫画的 **阅读顺序一致**，（例如 1.jpg, 2.jpg）

图片添加完成后，就可以前往 **设置页面** 填写 API 信息啦～(ง •̀_•́)ง

---

## 2️⃣ 设置页面

<img src="https://raw.githubusercontent.com/jedzqer/manga-translator/main/Tutorial/setting-screenshot.jpg" alt="设置页面" width="45%" />

本软件需要 **自备 API** 来进行漫画翻译 🛠️

如果你还不了解什么是 API，可以先在网上搜索相关资料，或者**询问AI**，获取一个可用的 API 服务。可以查找一些公益站来获取免费的模型。

设置步骤如下：

1. **填写 API 地址**

   * 通常是以 `/v1` 结尾的链接

2. **填写 API Key**

   * 🔒 请务必妥善保管
   * **不要泄露给他人**

3. **选择 / 填写模型名称**

   * ⚠️ 模型名称必须 **与供应商提供的完全一致**
   * 建议：

     * 直接从供应商官网复制模型名
     * 或点击 **「获取模型列表」** 按钮进行选择

*模型名称填写支持通过英文逗号分隔多模型以进行模型分流，例如：`A,A,B`（约 2/3 请求用 A，1/3 请求用 B）*

填写完成后就可以开始翻译啦～✨

---

## 3️⃣ 翻译页面 —— 开始翻译

<img src="https://raw.githubusercontent.com/jedzqer/manga-translator/main/Tutorial/translator-screenshot.jpg" alt="翻译页面" width="45%" />

翻译流程非常简单 💡：

1. 点击 **「批量上传」**，上传需要翻译的图片
2. 上传完成后，点击 **「翻译文件夹」**
3. 等待翻译完成 ⏳
4. 翻译结束后，点击 **「开始阅读」** 即可进入阅读页面 📖

*在全文速译开启的情况下，如果页数很多，建议先上传一部分，翻译完后再上传一部分。或者直接去设置里拉高API超时设置*

---

## 4️⃣ 阅读页面

在阅读页面中，你可以进行更精细的调整哦～(๑•̀ㅂ•́)و✧

* 🧩 **拖动翻译好的气泡**，调整到合适的位置

  * 软件会自动保存你的调整

* ❌ 如果 **某一页翻译效果不理想**：

  1. 回到漫画库
  2. 关闭 **「全文速译」** 开关
  3. 长按需要重新翻译的图片
  4. 选中后点击 **「重译」**

* ⚠️ 如果 **大量页面翻译效果不佳**：

  * 关闭 **「全文速译」** 后全部重译
  * 或切换到 **更智能 / 更高级的AI模型** 🤖✨

* 💬如果 **一些文字区域没有识别**：

  1. 点击阅读页面右上角的编辑按钮
  2. 点击加号按钮新建一个空白气泡
  3. 长按气泡调整气泡到合适大小
  4. 将空白气泡调整好大小和位置，完整覆盖到到未能识别的文字块上
  5. 点击右上角的绿勾退出编辑
  6. 等待气泡翻译

## 使用小技巧

- **跳转页面**：当你想要跳转到某一页面进行观看时，在漫画库的文件夹里面，点击对应的图片名称即可完成跳转
- **编辑气泡**：点击阅读页面右上角的编辑按钮进入编辑模式，长按气泡可以修改气泡大小或删除气泡，点击气泡可以编辑内容内容。点击右上角的加号可以新建气泡。
- **导出漫画**：如果你不喜欢在软件里面阅读，可以调整好翻译气泡后导出漫画，导出的漫画会在`手机储存/Pictures/manga-translate`目录内
